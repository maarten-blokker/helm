Home-net Helm Chart Repository

This is a Helm chart repository containing various helm charts for personal use.

How to Add and Search the Repository

        
          # Add this repo
          $ helm repo add home-net https://helm.home-net.app/helm
          "home-net" has been added to your repositories

          # Search for Charts with Helm 3+
          $ helm search repo home-net
